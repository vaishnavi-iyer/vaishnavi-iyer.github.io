HTML CSS and DOM -1
Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

W3C defines DOM (Document Object Model) as a platform and language-neutral interface that allows progeams and scripts to synamically access and update the content, structure, and style of a document.

What is the HTML DOM?
The HTML DOM is a standard object model and programming interface for HTML. It defines:
The HTML elements as objects
The properties of all HTML elements
The methods to access all HTML elements
The events for all HTML elements
In other words: The HTML DOM is a standard for how to get, change, add, or delete HTML elements.

Now, what DOM does is not something that we can view can directly however, the Dev Tools is a good interface to view the code after the DOM API has worked on it.

Now, "The House" expample is a classic example to explain this.

Imagine you are an architect, you made a design model of a house.

Now the structural elements of your model like the walls, windows and doors are what html is and the wall color of your model and the small decorative curatins is what css is. So now that the house model is ready, where is DOM and why hasn;t it come into the picture yet?

DOM comes into picture after your base is ready.

Now imagine this model is very flexible and you can move around the rooms of the model and and you can use ur poster to colors to paint the walls and fabric paint the decorative curtains and move around the staircase. Basically you are moving around the rooms like a jig saw puzzle to create that dream house of yours.... Well this run time renovation is what is DOM.

In case of a web page, runtime addtion and modification of your code is facilitated by DOM. All the interacive components of the web page like reading from a text box and detecting a mouse click is also the property of a DOM.

What is meant by boxifying design?

Each element that you see on a website has a rectangular outline to it. Visualising the digital design mockups in the form of boxes helps you define the structure of the html code.
Boxifying is essentially converting each element in the design mockup into rectangular boxes.

What is the box model?

The box model is a concept in CSS that defines the dimensions of each element in a page.
The box model includes:
1. Content in the centre
2. Padding
3. Border
4.Margin

The dimensions of each of these elements can be modified to achieve the desired layout of a webpage.
